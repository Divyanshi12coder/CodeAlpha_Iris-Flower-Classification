# 🌸 Iris Flower Classification – 3D Visualization

A machine learning project that classifies iris flowers into three species (*Setosa, Versicolor, Virginica*) using supervised learning algorithms. Includes **3D scatter plot visualization** for realistic feature exploration.

---

## 📌 Project Overview
- Dataset: Famous **Iris dataset** (150 samples, 4 features)
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target Classes:
  - Iris Setosa
  - Iris Versicolor
  - Iris Virginica

This project demonstrates:
- Data preprocessing
- Model training & evaluation
- 3D visualization of feature relationships
- Deployment‑ready classification pipeline
